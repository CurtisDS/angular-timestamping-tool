# Angular Timestamping Tool

An app to allow you to create timestamps

### Kepboard shortcut:

`Numpad+` - Insert a split (*must not be focused in the time adjust input field*)

`Ctrl`+`Z` - Undo

`Ctrl`+`Y` - Redo
